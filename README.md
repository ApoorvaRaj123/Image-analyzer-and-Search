### Prerequisites

**Node version 18.17 or later**

### Fill in API keys

**Open AI**

```js
 headers: {
     // you api key here
     "OPENAI_API_KEY ": " www.openai.com",
   }
```

### Install packages

```shell
npm i
```

### Setup .env file

```js
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=
// From Uploading Thing docs

// or fill in Rapid api keys here
OPENAI_API_KEY =
```

### Start the app

```shell
npm run dev
```
